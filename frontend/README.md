## Getting Started

### Requipments
- [https://nodejs.org/en](node) - Runtime

### Fast way
Run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Slow way

```bash
npm run build && npm run start
# or
pnpm build && pnpm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Running tests

```bash
npm run test
# or
pnpm test
```
