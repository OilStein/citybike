## Getting Started

### Requipments
- [Node](https://nodejs.org/en) - Runtime

### Running application
Run the unoptimized development server:

```bash
npm run dev
# or
pnpm dev
```
or run optimized

```bash
npm run build && npm run start
# or
pnpm build && pnpm start
```

Open <http://localhost:3000> with your browser.

## Running tests

```bash
npm run test
# or
pnpm test
```
